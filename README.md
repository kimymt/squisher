# Squisher

iPhone の写真をその場で圧縮 — **写真は端末から出ません**。

🔗 **https://squisher.mymt.casa**

## できること

- 写真を選んで、品質を選んで、保存するだけ
- 3 段階の品質プリセット(高品質 / 標準 / 強力圧縮)
- 複数ファイルを一括で処理
- ファイルごとに JPEG / WebP を切替
- サイズが増えたファイルは自動で除外
- ホーム画面に追加すれば、ふつうのアプリのように起動

## 使い方

1. **写真を選ぶ** — 空のカードをタップして、カメラロールから写真を選択
2. **品質を選ぶ** — 画面上部のボタンから「高品質 / 標準 / 強力圧縮」をタップ
3. **保存する** — 画面下部の「写真に保存」をタップ → iOS の共有シートから写真アプリへ

品質を切り替えると、すでに処理済みのファイルも自動で再圧縮されます。

## 動作環境

### おすすめ: **iOS 18 以降の Safari**(iPhone)

- HEIC を含むすべての写真形式に対応
- 写真アプリへ直接保存(共有シート → 画像を保存)
- ホーム画面に追加すれば、画面いっぱいに表示 + 専用アイコン

### その他のブラウザ(パソコンの Chrome / Firefox / Android など)

- **JPEG / PNG のみ**対応
- HEIC は処理できません(iPhone 以外のブラウザは HEIC を読み込めない仕組みのため)
- 保存先は「ダウンロード」フォルダになります

最低要件: iOS 16.4 以降(写真アプリへの保存が安定して動作する最低バージョン)。それより古い iOS では、写真アプリではなくダウンロードフォルダに保存されます。

## 注意事項

### HEIC は iPhone の Safari でのみ対応

iPhone で撮影した写真の多くは **HEIC** という形式ですが、HEIC を直接扱えるのは iPhone の Safari だけです。

- ✅ **iPhone の Safari**: HEIC を選んでも、iOS が自動で JPEG に変換して Squisher に渡します。利用者は HEIC を意識する必要はありません。
- ❌ **パソコンや Android のブラウザ**: HEIC を選ぶと「このブラウザは HEIC に対応していません」というエラーがファイルごとに表示されます。事前に JPEG / PNG に変換するか、iPhone でアクセスしてください。

### 写真は端末から出ません

すべての圧縮処理はあなたの端末内で完結します。

- 写真をサーバーへ送ることはありません
- 閲覧履歴の収集や利用者の追跡はしていません
- インターネットへの通信は初回のアプリ取得のときだけ(2 回目以降はインターネットに繋がっていなくても動きます)

### 元の写真は変更されません

圧縮結果は新しいファイルとして保存されます。カメラロールの元の写真はそのまま残ります。

### サイズが増えたファイルは保存しない

すでに圧縮されたファイルをもう一度圧縮すると、場合によっては逆にサイズが大きくなることがあります。Squisher はそのようなファイルを保存リストから自動で除外します(画面下部のスイッチで無効化できます)。

## ホーム画面に追加(おすすめ)

ホーム画面に追加すると、画面いっぱい + 専用アイコンで「ふつうのアプリ」として起動できます。

1. Squisher を Safari で開く
2. 共有ボタン(□↑)をタップ
3. 「ホーム画面に追加」を選択

初回の圧縮が成功した直後に、画面内に追加手順のヒントが表示されます(× で閉じれば、二度と出ません)。

## よくある質問

**Q. どれくらい小さくなりますか?**
A. 写真や品質設定にもよりますが、iPhone で撮った写真なら標準プリセットで **40–60% 削減**が目安です。Squisher は保存前の画面で 1 枚ずつ削減率を表示します。

**Q. 動画は圧縮できますか?**
A. 現在は画像のみです(動画は今後の対応予定)。

**Q. 何枚まで処理できますか?**
A. 上限は設定していませんが、iPhone のメモリ容量によります。実用上は **20–50 枚程度**が快適です。

**Q. インターネットに繋がっていなくても使えますか?**
A. はい。一度アクセスしておけば、機内モードでも動作します。

**Q. 写真が「写真」アプリに保存されません**
A. iOS 18 以降 + ホーム画面に追加した状態がもっとも安定します。iOS 16.4〜17 では共有シートが出ますが、機種・OS によってはダウンロード扱いになることがあります。

**Q. 古い見た目のまま更新されません**
A. Safari の設定で「このサイトのデータを削除」を実行してから再度アクセスしてください。
